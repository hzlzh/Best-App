## 购买建议

* 建议尽量选择`冰点价格`或`打折促销`时购入，可以获得更大的优惠
* 部分软件开发商会有**官方**`团购打折`
* `家庭装促销包授权`适合家里人一起使用
* 关注和参与`个人`或`集体`组织的团购`活动`
* 选择购买`Mac App Store`的版本通常升级后续版本会 Free
* 如果官方有非沙盒版，建议不要选择`Mac App Store`的阉割版，购入原版
